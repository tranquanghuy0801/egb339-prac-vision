# EGB339 Introduction to Robotics

The prac computer vision and kinematics assignments using the Machine Vision and Robotics Toolboxes of Peter Corke (http://petercorke.com/wordpress/) for EGB339 - Introduction to Robotics 

The computer vision assignment is about to analyse the taken images including detecting the colour, size, number of blobs and computing th homography of blobs on the image  

The kinematics assignment is about to move the tooltip of Orion Robot Arm to the required positions on the image
